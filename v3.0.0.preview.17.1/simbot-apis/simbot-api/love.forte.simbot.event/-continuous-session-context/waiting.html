val session: ContinuousSessionContext = ...

session.waiting { provider -> // this: EventProcessingContext
   // ...
   provider.push(...)
}

超时处理

使用 withTimeout 或其衍生函数来进行超时控制。

val session: ContinuousSessionContext = ...

withTimeout(5.seconds) {
   session.waiting { provider -> // this: EventProcessingContext
      // ...
      provider.push(...)
   }
}
