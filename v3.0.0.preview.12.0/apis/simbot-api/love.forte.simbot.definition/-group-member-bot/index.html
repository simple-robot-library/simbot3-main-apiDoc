GroupMemberBot

interface GroupMemberBot : MemberBot, GroupMember
Content copied to clipboard

一个 Bot 在一个 群 中所扮演的 成员 对象。

GroupMemberBot 同时实现 MemberBot 和 GroupMember, 通过 Group.bot 获取并代表此bot在目标群中的成员身份。
