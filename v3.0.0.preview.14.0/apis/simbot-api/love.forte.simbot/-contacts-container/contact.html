abstract suspend fun contact(id: ID): Contact?
Content copied to clipboard

通过唯一标识获取对应的 Contact 实例。当且仅当因标识对应联系人不存在而导致无法获取时得到null。

在一些组件支持的情况下，contact 会在目标联系人会话不存在的时候尝试主动创建会话。 假若这个过程中发生了预期内的错误（例如网络请求错误、id 格式错误等）时， 不会视为"未找到联系人"，因此会抛出对应异常而不是返回null。
