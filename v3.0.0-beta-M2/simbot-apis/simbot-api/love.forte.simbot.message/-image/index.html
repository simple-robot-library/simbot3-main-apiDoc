一个图片消息。

Image 比较常见的有如下几种形式：

  1. 由客户端本地上传。此类 Image 多为使用 资源信息 上传。

  2. 客户端接收到的消息。此类 Image 大多数持有一个由服务端所提供的 唯一标识。

标准消息类型中提供了一个 ResourceImage 来实现上述两种情况中的第 1 种情况： 由客户端提供 Resource 的图片类型。
