GroupBot

interface GroupBot : OrganizationBot
Content copied to clipboard

一个 Bot 在一个 群 中所扮演的对象。

GroupBot 实现 OrganizationBot, 并对外提供转化为 Member 的能力, 以允许将自身转化为在目标群中的成员。
