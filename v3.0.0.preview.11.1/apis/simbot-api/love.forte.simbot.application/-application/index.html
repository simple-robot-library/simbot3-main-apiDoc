Simple Robot 的应用.

一个 simbot应用 代表了已经配置完毕的运行时作用域，此作用域内包含了有关事件处理、事件推送的相关内容.

每个 Application 中，应当包括一个 事件处理器 、一些 bot管理器 和一些注册的组件。
