配合 Event.Key.invoke 并标记其构建的监听函数为异步的。

FooEvent { event: FooEvent -> // this: EventListenerProcessingContext
  // do handle

  EventResult.defaults()
}.async(true)

// or

FooEvent { event ->
   // ...
   eventResult()
}.async() // default param: true

   // ...
