Objectives

interface Objectives : BotContainer, IDContainer
Content copied to clipboard

Objectives 是对与 Bot 相关联的对象 （一个组织或一个具体的用户） 的统称。

不论 组织 还是 用户，它们均来自一个 Bot.

Objectives 本身仅代表这个对象的概念，不能保证其本身拥有 发送消息 的能力。
