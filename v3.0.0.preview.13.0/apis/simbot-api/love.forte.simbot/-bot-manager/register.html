通过 BotVerifyInfo 注册并管理一个 Bot.

BotVerifyInfo 通常情况下是由组件读取 *.bot 文件并提供给当前 BotManager, 作为用户一般情况下不需要使用函数。

BotManager 的实现通常需要提供其他额外的友好api来使用。
