基础的 boot-core 所使用的启动类标记注解，提供部分参数。

不应 在spring环境下使用。

此注解在当前环境 SimbootEntrance 所选取的启动入口为 CoreBootEntrance 的情况下有效。

Parameters

scanPackages

指定所需扫描包路径。 如果为空默认选取标记类所在包路径为根路径。

topListenerScanPackages

指定需要扫描的顶层监听函数路径。顶层监听函数正常来讲只有kotlin下会有，例如：

@Listener
suspend fun FriendEvent.onEvent() { ... }
