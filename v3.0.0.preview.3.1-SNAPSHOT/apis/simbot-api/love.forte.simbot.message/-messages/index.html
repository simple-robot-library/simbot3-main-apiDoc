Messages

interface Messages : List<Message.Element<*>> , RandomAccess, Message
Content copied to clipboard

消息列表，即 MsgElement 的列表，连接多条消息元素的链表。

Messages 是不可变的，但是 Messages 中的元素并不一定。每次进行 plus 都应视为得到了一个新的 Messages 实例。

Messages 中可以存在多个不同组件之间的 MsgElement, 但是除了与 SimbotComponent 混用之外，不建议这么做。 大多数情况下，组件对于 Messages 的解析很少会顾及到其他组件，而当遇到不支持的组件的时候，大概率会将其忽略或抛出异常。
