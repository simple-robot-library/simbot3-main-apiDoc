一个 行为动作。

不论是对于 love.forte.simbot.Bot, 组织 还是 用户 等，

尽管他们在不同组件中的能力与含义有所不同，但是它们总会存在一部分相似的能力与行为, 例如对于一个群成员， 在权限足够的情况下，有很多平台都允许对其进行禁言或踢出。

Action 对这些常见的行为进行描述，假若组件存在支持的相似功能，优先考虑实现相关接口。

Action 应当完全有组件实现者实现，并提供所有对应类型的预先定义。
