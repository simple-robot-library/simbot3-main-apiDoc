abstract class BotManager<B : Bot> : BotRegistrar, ComponentContainer, Survivable

Bot 管理器。 BotManager 应当是 获取、注册 Bot 的唯一公开途径， 所有 BotManager 均由 OriginBotManager 进行管理。

BotManager 实现 Survivable, 其存活周期与 Bot 无关。
